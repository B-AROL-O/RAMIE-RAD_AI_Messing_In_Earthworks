# TELEOPERATION

Applications that use direct speed commands to pilot the robot

## CONSOLE TELEOPERATION

It's a python application that when wasd are pressed move the motor, no press motor stopped, q exit the application

It runs by going SSH into the robot

Keyboard library requires root permissions, so discarded, I can go without apparently, it's hacky but works
