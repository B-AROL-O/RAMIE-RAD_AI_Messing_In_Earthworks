# Luckfox PicoKVM

I got a cheap and compact KVM that connects via ethernet HDMI and USB

## Buy

[Ebay 49€](https://www.ebay.it/itm/267432348798)

## Electrical Connections

- HDMI to SBC: this is the video out
- USB to SBC: this is power from SBC and keyboard and mouse to SBC
- Ethernet: to host. This has the webserver where the KVM can be accessed with remote screen and other functionalities
