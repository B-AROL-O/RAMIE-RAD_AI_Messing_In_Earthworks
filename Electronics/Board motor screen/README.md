# Motor Board

This is a board that I designed in 2012 to control robots

It was convenient for me to use this board as I have a bounch of them assembled

It has an Atmel AT324PA PDIP40, an ISP programmer, power and breaks out all the pins

I also have a more recent board based on Atmel 4809 in SMD package, both work for this application.
